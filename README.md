# FavoGit

Un buscador de Usuarios de GitHub que te permite marcar como favoritos los Usuarios que mas te gusten!!

## Servidor de Desarrollo

Para probar la aplicación, ejecuta `ng serve` y navega a `http://localhost:4200/`.
